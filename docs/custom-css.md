# Custom CSS Notes

This document explains the role of custom CSS in the portfolio website.

## Purpose

Custom CSS is used to improve the visual style of specific website sections and make them match the dark design of the portfolio.

The main goal is to keep the website clean, readable and professional on desktop, tablet and mobile screens.

## Areas improved with CSS

1. Review section styling
2. Dark theme adjustments
3. Text color and contrast
4. Form spacing
5. Mobile layout fixes
6. Button and section alignment

## Review section

The review section was styled so that it fits the dark theme of the website.

The design focuses on readable text, clear spacing, rounded cards and a simple professional appearance.

## Mobile improvements

Some sections need special attention on smaller screens.

The goal is to avoid text being cut off, cards becoming too narrow or buttons moving outside the visible area.

## CSS approach

The CSS should stay simple and easy to maintain.

Large changes should be tested first on desktop and then checked again on mobile view.

## What should not be included here

This repository should not include private WordPress configuration files, plugin files with private settings or full hosting backups.

Only clean notes, reusable CSS snippets and project documentation should be added.
