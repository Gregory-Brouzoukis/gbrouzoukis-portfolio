# SEO Notes

This document keeps track of the main SEO work and indexing setup for the portfolio website.

## Website address

https://gbrouzoukis.gr

## Main SEO goal

The goal is to help the website appear correctly on Google when someone searches for Gregory Brouzoukis, the portfolio website or related technical work.

The website should present a clean title, a correct description, the right logo and only the pages that are useful for search results.

## Google Search Console

Google Search Console is used to verify the domain and monitor indexing.

The domain property setup is useful because it can cover the full domain and not only one version of the website.

## Pages that should appear on Google

1. Home page
2. Portfolio or professional profile pages
3. Project pages when they are ready
4. Useful public pages with real content

## Pages that should not appear as search results

1. Privacy policy
2. Cookie policy
3. Terms of use
4. Temporary blog pages without finished content
5. Empty or low value pages

These pages can still exist on the website, but they do not need to appear as Google results.

## Current indexing direction

The website should stay accessible to users, while unnecessary pages can be set to noindex.

This keeps the site cleaner in search results and helps Google focus on the most important pages.

## Logo and appearance in search

The logo and favicon setup should be checked through the website settings, structured data and Google Search Console.

Google may take time to update how the site appears in search results, even after the correct changes have been made.

## Future improvements

1. Add stronger page titles
2. Improve meta descriptions
3. Add project pages with useful text
4. Add screenshots and real examples
5. Keep the sitemap clean
6. Review indexing after every major change
