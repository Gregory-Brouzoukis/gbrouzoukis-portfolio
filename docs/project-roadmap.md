# Project Roadmap

This document describes the next steps for the portfolio website.

## Current stage

The website is live and the basic professional structure is already in place.

The main focus now is to improve the content, make the project pages stronger and connect the website more clearly with GitHub.

## Next improvements

1. Add better project descriptions
2. Add screenshots from the live website
3. Improve mobile layout where needed
4. Add more programming projects
5. Improve the contact section
6. Keep SEO settings clean
7. Review Google indexing after updates

## Project pages

Each project page should explain what the project does, why it was created, what tools were used and what was learned during the process.

This will make the portfolio more useful for people who want to understand the work behind each project.

## GitHub connection

The GitHub profile should support the website by showing real projects, clear descriptions and organized repositories.

The portfolio website and GitHub profile should work together as one professional online presence.

## Long term direction

The long term goal is to build a stronger technical profile with projects in web development, Python, applications and AI automation.

The website can grow over time as more work is completed.
